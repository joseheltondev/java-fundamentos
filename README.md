# Java Fundamentos

Repositório com exercícios e projetos desenvolvidos durante meus estudos da linguagem Java.

## Objetivos

- Aprender os fundamentos da linguagem Java;
- Desenvolver lógica de programação;
- Praticar Programação Orientada a Objetos (POO);
- Construir um portfólio para estágio e futuras oportunidades na área de desenvolvimento.

## Tecnologias Utilizadas

- Java
- Spring Tools Suite (STS)
- Git
- GitHub
